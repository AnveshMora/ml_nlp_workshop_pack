# Mini Project Evaluation Rubric (100 points)

## 1) Problem framing (10)
- Clear task definition (binary/multi-class)
- Dataset described properly

## 2) Data prep quality (15)
- Correct train/test split
- Good cleaning/handling missing values
- No obvious leakage

## 3) Baseline model (15)
- Simple baseline exists and runs end-to-end
- Metrics shown

## 4) Performance & metrics (20)
- Uses F1 + confusion matrix
- Correct interpretation

## 5) Improvements / iteration (15)
Any one improvement:
- TF-IDF vs BoW
- n-grams tuning
- model comparison
- class weights / threshold tuning

## 6) Error analysis (15)
- Shows misclassified examples
- Explains patterns and failure modes

## 7) Demo readiness (10)
- Custom text prediction cell works
- 5 sample predictions shown

### Bonus (+10)
- Transformer comparison
- Save/load pipeline
- Simple deployment (CLI/Streamlit)
